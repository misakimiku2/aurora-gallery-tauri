## 目标
保持 `.aurora` 文件扩展名不变，内部使用 ZIP 格式存储，嵌入图片二进制数据并保存画布视图状态。

## 具体修改

### 1. 添加依赖 (`package.json`)
安装 `jszip` 库

### 2. 修改保存逻辑 (`src/components/ImageComparer.tsx`)
- 创建 `JSZip` 实例
- 添加 `manifest.json`: 版本(2.0)、创建时间、会话名称
- 添加 `viewport.json`: 画布缩放(scale)、位移(x, y)
- 添加 `layout.json`: 图片布局信息、注释、层级顺序
- 创建 `images/` 文件夹，将每个图片以原始二进制格式存入
- 生成 ZIP 数据并保存为 `.aurora` 文件

### 3. 修改加载逻辑 (`src/components/ImageComparer.tsx`)
- 读取 `.aurora` 文件为二进制
- 使用 `JSZip.loadAsync` 解压
- 读取各 JSON 文件恢复状态
- 从 `images/` 读取图片数据创建 Blob URL
- 向后兼容旧版本纯 JSON 格式

### 4. ZIP 内部结构
```
文件名: xxx.aurora (实际是 ZIP)
├── manifest.json
├── viewport.json
├── layout.json
└── images/
    ├── {id1}.png
    └── ...
```

## 优势
- 文件扩展名保持 `.aurora` 不变
- 图片以原始二进制存储，无 Base64 膨胀
- ZIP 压缩减小体积
- 画布状态完整保存
- 向后兼容
