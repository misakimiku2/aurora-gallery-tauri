## 问题

图片加载完成后 Canvas 不自动重绘，必须手动交互（滚轮/点击）才显示。

## 根本原因

`img.onload` 回调中只更新了 `loadedCount`，但 `drawCanvas` 的 `useCallback` 依赖项中没有包含 `loadedCount`，导致图片加载完成后不会触发重绘。

## 修复方案

将 `loadedCount` 添加到 `drawCanvas` 函数的 `useCallback` 依赖数组中。

## 代码修改

文件：`src/components/ImageComparer.tsx`

修改第 621-711 行的 `drawCanvas` 函数，将 `loadedCount` 加入依赖数组：

```typescript
const drawCanvas = useCallback(() => {
  // ... 现有代码不变
}, [transform, containerSize, isDarkMode, activeImageIds, zOrderIds, layoutItemMap, getVisibleItems, loadedCount]);
```
