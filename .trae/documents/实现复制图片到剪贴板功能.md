## 实现计划：复制图片到剪贴板功能

### 功能描述
用户在图片上右键点击时，可以选择"复制图片"，将图片数据复制到系统剪贴板，然后可以在其他应用（如微信、QQ、Photoshop等）中粘贴使用。

### 实现步骤

#### 1. Rust 后端 (Tauri)
- **添加依赖**: 在 `Cargo.toml` 中添加 `arboard` 库（跨平台剪贴板库）
- **新建命令**: 在 `main.rs` 中添加 `copy_image_to_clipboard` 命令
- **功能实现**: 读取图片文件并写入系统剪贴板

#### 2. 前端 TypeScript
- **API Bridge**: 在 `tauri-bridge.ts` 中添加 `copyImageToClipboard` 函数
- **右键菜单**: 在 `ContextMenu.tsx` 中添加"复制图片"菜单项（仅对单张图片显示）
- **App.tsx**: 实现 `handleCopyImageToClipboard` 函数，调用 bridge 并显示提示

#### 3. 国际化
- 翻译键已存在：`copyImage` 和 `imageCopied`

### 需要修改的文件

| 文件 | 修改内容 |
|------|----------|
| `src-tauri/Cargo.toml` | 添加 `arboard = "3"` 依赖 |
| `src-tauri/src/main.rs` | 添加 `copy_image_to_clipboard` 命令 |
| `src/api/tauri-bridge.ts` | 添加 `copyImageToClipboard` 函数 |
| `src/components/ContextMenu.tsx` | 添加"复制图片"菜单项 |
| `src/App.tsx` | 实现 `handleCopyImageToClipboard` 函数并传递给 ContextMenu |

### 技术方案

使用 `arboard` 库实现跨平台剪贴板操作：
- Windows: 直接写入剪贴板
- macOS: 使用 NSPasteboard
- Linux: 使用 X11/Wayland 剪贴板
