## 优化目标
在进入和退出参考模式时，在下方显示提示消息

## 实现方案

由于 `ImageComparer` 组件是独立渲染的，我将在组件内部使用 `useToasts` hook 来显示提示消息。

### 修改内容

1. **导入 `useToasts` hook**
   ```typescript
   import { useToasts } from '../hooks/useToasts';
   ```

2. **在组件中使用 `useToasts`**
   ```typescript
   const { toast, showToast } = useToasts();
   ```

3. **在进入参考模式时显示提示**
   ```typescript
   showToast('解除窗口大小限制。', 2000);
   ```

4. **在退出参考模式时显示提示**
   ```typescript
   showToast('开启窗口大小限制。', 2000);
   ```

5. **在组件 JSX 中添加 Toast UI**
   在组件底部添加一个简单的 toast 显示元素

## 代码变更位置
- 文件：`src/components/ImageComparer.tsx`
- 添加 `useToasts` 导入
- 在组件内部使用 `useToasts`
- 在 `toggleReferenceMode` 函数中调用 `showToast`
- 在 JSX 中添加 toast 渲染
