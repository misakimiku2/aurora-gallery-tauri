## 计划概述

在 TreeSidebar.tsx 的"所有标签"栏下方新增一个"画布"栏，点击后创建新标签页并打开空的图片对比组件。

## 实现步骤

### 1. 修改 TreeSidebar.tsx

在 TagSection 组件之后添加一个新的 CanvasSection 组件：

```tsx
interface CanvasSectionProps {
  onOpenCanvas: () => void;
  t: (key: string) => string;
  isSelected?: boolean;
}

const CanvasSection: React.FC<CanvasSectionProps> = React.memo(({ onOpenCanvas, t, isSelected }) => {
  return (
    <div className="select-none text-sm text-gray-600 dark:text-gray-300 relative">
      <div 
        className={`flex items-center py-1 px-2 cursor-pointer transition-colors border border-transparent group relative mt-2 ${isSelected ? 'text-white border-l-4 shadow-md' : 'hover:bg-gray-200 dark:hover:bg-gray-800'}`}
        style={isSelected ? { backgroundColor: '#10b981', borderLeftColor: 'rgba(16,185,129,0.32)' } : undefined}
        onClick={onOpenCanvas}
      >
        <div className="p-1 mr-1 rounded opacity-0"></div>
        <div className="flex items-center flex-1">
          <Layout size={14} className={`mr-2 ${isSelected ? 'text-white' : 'text-emerald-500 dark:text-emerald-400'}`} />
          <span className={`font-bold text-xs uppercase tracking-wider transition-colors ${isSelected ? 'text-white' : 'text-gray-500 dark:text-gray-400 group-hover:text-black dark:group-hover:text-white'}`}>{t('sidebar.canvas')}</span>
        </div>
      </div>
    </div>
  );
});
```

在 Sidebar 组件的 props 中添加 `onOpenCanvas` 回调，并在 TagSection 之后渲染 CanvasSection。

### 2. 修改 App.tsx

添加一个新的回调函数 `handleOpenCanvas`，它会创建一个新的标签页并设置 `isCompareMode: true` 和空的 `selectedFileIds`：

```tsx
const handleOpenCanvas = useCallback(() => {
  const newTab: TabState = {
    ...DUMMY_TAB,
    id: Math.random().toString(36).substr(2, 9),
    folderId: state.roots[0] || '',
    selectedFileIds: [], // 空的图片对比
    isCompareMode: true,
    history: {
      stack: [{
        folderId: state.roots[0] || '',
        viewingId: null,
        viewMode: 'browser',
        searchQuery: '',
        searchScope: 'all',
        activeTags: [],
        activePersonId: null
      }],
      currentIndex: 0
    }
  };
  setState(prev => ({ ...prev, tabs: [...prev.tabs, newTab], activeTabId: newTab.id }));
}, [setState, state.roots]);
```

### 3. 传递回调给 Sidebar

在 App.tsx 中 Sidebar 组件的调用处添加 `onOpenCanvas={handleOpenCanvas}` 属性。

### 4. 添加翻译

在 translations.ts 中添加 `sidebar.canvas` 的翻译（中文："画布"，英文："Canvas"）。

## 文件修改清单

1. `src/components/TreeSidebar.tsx` - 添加 CanvasSection 组件和 onOpenCanvas prop
2. `src/App.tsx` - 添加 handleOpenCanvas 回调并传递给 Sidebar
3. `src/utils/translations.ts` - 添加画布翻译（可选，如果翻译已存在则跳过）
