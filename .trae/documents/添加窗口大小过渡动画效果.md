## 优化目标
为退出参考模式时的窗口大小恢复添加平滑过渡动画效果

## 当前问题
退出参考模式时，窗口从小的尺寸突然变成 1280x800，没有过渡效果，显得生硬。

## 解决方案

由于 Tauri 的 `window.setSize()` API 本身不支持动画，我们需要自己实现一个动画循环来连续更新窗口大小。

### 实现步骤

1. **创建窗口大小动画函数**
   在 `toggleReferenceMode` 函数中添加一个内部动画函数，使用 `requestAnimationFrame` 实现平滑过渡。

2. **动画参数**
   - 动画时长：约 300ms
   - 缓动函数：使用 ease-out 效果，让动画更自然
   - 帧率：使用 `requestAnimationFrame` 自动适配显示器刷新率

3. **具体实现**

```typescript
// 在退出参考模式时，使用动画过渡窗口大小
const animateWindowResize = async (
  targetWidth: number,
  targetHeight: number,
  duration: number = 300
) => {
  const window = getCurrentWindow();
  const startSize = await window.innerSize();
  const startWidth = startSize.width;
  const startHeight = startSize.height;
  const startTime = performance.now();

  // ease-out 缓动函数
  const easeOutCubic = (t: number) => 1 - Math.pow(1 - t, 3);

  return new Promise<void>((resolve) => {
    const animate = (currentTime: number) => {
      const elapsed = currentTime - startTime;
      const progress = Math.min(elapsed / duration, 1);
      const easedProgress = easeOutCubic(progress);

      const newWidth = Math.round(startWidth + (targetWidth - startWidth) * easedProgress);
      const newHeight = Math.round(startHeight + (targetHeight - startHeight) * easedProgress);

      window.setSize(new LogicalSize(newWidth, newHeight));

      if (progress < 1) {
        requestAnimationFrame(animate);
      } else {
        resolve();
      }
    };

    requestAnimationFrame(animate);
  });
};
```

4. **修改退出参考模式的逻辑**
   将原来的直接设置窗口大小改为调用动画函数：

```typescript
// 原来：
await window.setSize(new LogicalSize(1280, 800));

// 改为：
await animateWindowResize(1280, 800, 300);
```

## 预期效果
退出参考模式时，窗口大小会从小尺寸平滑过渡到 1280x800，动画持续约 300ms，使用 ease-out 缓动曲线，让动画看起来更自然。
